# Agent Notes

This repo is a web-based idle game: **Progress Knight Quest Booosted**, a boosted continuation of Progress Knight Quest with multiple prestige layers.

Use **ponytail** for coding work in this project:

- Prefer the simplest solution that works.
- Reuse existing HTML, CSS, and JavaScript patterns before adding new ones.
- Avoid new dependencies unless there is a clear need.
- Keep changes small and focused on the requested behavior.
- Skip speculative abstractions, scaffolding, and "for later" code.
